Abstract:
This shapefile is derived from GIS data that was originally developed for Colorado's Western Slope in the mid-1990's as part of the CDSS project. Any subsequent data processing is described in the Data Quality Information section below. The release notes that were originally created for the data are as follows:

This directory contains the Arc/Info coverages populated for CRDSS as part of Task 1.06 and 2.03 Spatial Database Development for cities. This coverage contains locations for municipalities for the Western Slope of Colorado. The 1990 census data for populations is stored with the GRASS coverage Municipalities.

Purpose: Spatial data in support of the CDSS project.

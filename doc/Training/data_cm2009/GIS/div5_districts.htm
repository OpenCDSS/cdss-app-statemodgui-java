Description:
Abstract:
This spatial data defines the Colorado Division of Water Resources (DWR) Water District Boundaries. District boundaries are administrative boundaries set by the State Engineer, which are based primarily on stream drainage systems.
Purpose:
Spatial data integration for the Colorado Division of Water Resources
Supplemental_Information:
The source data is State Engineer's Office Division Map (1:2,000,000 ). In addition, where appropriate, the East Slope Data was modified using USGS County Maps (1:50,000) and the West Slope Data was modified using USGS Hydrologic Unit Coverage Maps (HUC, 1:100,000). Any district boundary modification were completed using 1:24000 DRGs or topographic maps.

Process_Step:
Process_Description:
The creation of this spatial data set involved firstly digitizing the State Engineer Office's 1: 2,000,000 Division Map: District boundaries are administrative boundaries set by the State Engineer, which are based primarily on stream drainage systems. Secondly, where appropriate, the east slope data was further modified using USGS County Maps (1:50,000) and the West Slope Data was modified using USGS Hydrologic Unit Coverage Maps (HUCs, 1:100,000). The USGS maps delineate the basin boundaries based on elevation and hydrography. Division Offices and District Water Commissioners assisted with QA/QC work. Any subsequent district boundary modifications made to spatial dataset used 1:24,000 USGS DRGs or topographic maps as a base layer. Specifically, Division 5 boundaries were refined by Brian Romig to follow 1:24,000 ridgelines (4/02).

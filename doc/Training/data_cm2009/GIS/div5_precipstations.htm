Process_Step:
Process_Description:
Data were created from relational data in HydroBase database version 20100812 using a database query that generates a table of the precipitation station records in the correct division. The coordinates of the points are stored in the table's fields and are used to create a point shapefile. Note that this shapefile was created by selecting the HydroBase records whose "DIV" field shows the station as belonging to this division. The points' locations are derived from the coordinates stored in the table and as such, may fall outside the boundaries of this division.
